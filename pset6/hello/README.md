# Hello

This program revisit Problem Set 1's Hello and does the exact same thing, but this time in Python. Hello is an extremely simple program that greets the user by their name.

## Implementation Details

* The program prompts the user for their name, then greets them by it.

## Usage

To run the program, make sure to be in the `hello` directory, execute `python3 hello.py` and the program's behaviour should be as per the examples below.

```bash
$ python hello.py
What is your name?
David
hello, David
```

```bash
$ python hello.py
What is your name?
Emma
hello, Emma
```
